Case Study

ReAppraise

A SwiftUI prototype for photo-based item valuation.

Problem

Used-item pricing often requires manual searches across comparable listings. ReAppraise organizes that workflow around a photo, item details, and condition selection.

What I Built

I built the SwiftUI interface, image picker flow, Firebase callable-function wrappers, category and condition controls, comparable listing display, and median-based estimator.

Implementation

The app cleans OCR text, labels, logos, category hints, and user input into concise search queries, then calculates low, median, and high estimates with condition multipliers.

Prototype Limits

Live estimates require Firebase callable functions and real Firebase configuration. This should be presented as a prototype, not a deployed app.
