ReAppraise

A SwiftUI prototype for photo-based item valuation.

Problem

Used-item pricing often requires manual searches across comparable listings. ReAppraise organizes that workflow around a photo, item details, and condition selection.

What I Built

I built the SwiftUI interface, image picker flow, Firebase callable-function wrappers, category and condition controls, comparable listing display, and median-based estimator.

Implementation

The app cleans OCR text, labels, logos, category hints, and user input into concise search queries, then calculates low, median, and high estimates with condition multipliers.

Prototype Limits

Live estimates require Firebase callable functions and real Firebase configuration. This should be presented as a prototype, not a deployed app.

Case Study

Bean

A native iOS app for discovering independent coffee shops and matcha bars nearby.

Problem

General map searches often mix independent specialty shops with large chains and unrelated results. Bean focuses discovery on local coffee and matcha destinations.

What I Built

I designed and developed the SwiftUI interface, interactive MapKit search, shop filters, place details, Apple Maps directions, location sharing, favorites, and user preferences.

Location & Search

MapKit and Core Location support current-location, address, and visible-map searches. Results are filtered and deduplicated to prioritize independent specialty shops.

Networking

Async/await requests retrieve live OpenStreetMap data from the Overpass API. Multiple endpoint fallbacks, Codable parsing, local caching, and loading, error, and empty states make searches more resilient.

Persistence & State

UserDefaults and AppStorage preserve complete favorite-place records and preferences across launches, while observable shared services coordinate location permissions, search results, and map state.

Interface Architecture

I created reusable colors, typography, cards, animations, view modifiers, and modular components to keep the interface consistent. The app is built entirely with native Apple frameworks and has no external dependencies.
