Case Study

ReAppraise

A mobile valuation workflow for turning an item photo into a cleaner search query, comparable listings, and a condition-adjusted price range.

Problem

Figuring out what used items might be worth can slow down resale, donation, and cleanup decisions. ReAppraise reduces the manual research needed before deciding whether to sell, donate, or keep an item.

What I Built

I built a SwiftUI iOS prototype with photo capture, image picker integration, Firebase callable-function wrappers, category and condition controls, comparable listing display, and a median-based price estimator.

Implementation

The app cleans OCR text, logos, labels, web entities, category hints, and user-entered details into concise search queries, then calculates low, median, and high estimates adjusted by condition multipliers.

Prototype Limits

The iOS code is present, but live estimates require Firebase callable functions and real Firebase configuration. The backend source is not included, so this should be presented as a prototype rather than a deployed app.

Case Study

Dulcet & Delish Cafe Website

A lightweight multi-page restaurant site built to make menus, hours, ordering, and contact information easy to find.

Problem

The cafe needed a clear online presence where visitors could browse current offerings, confirm hours and location details, and access online ordering without relying only on third-party listings or social media.

What I Built

I built a plain HTML, CSS, and JavaScript site with homepage, menu, and contact pages. The pages use warm branding, structured restaurant content, responsive navigation, a contact form, and embedded location directions.

Implementation

The site includes semantic sections, CSS custom properties, responsive breakpoints, accessible mobile navigation, SEO meta tags, Open Graph metadata, and JSON-LD local business schema.

Future Improvements

Next steps include adding a hosted live URL, improving mobile menu readability with zoomable or text-based menus, optimizing images, and adding more controlled contact form success and error handling.
