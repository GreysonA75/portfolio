# Greyson Alegria Portfolio

A modern, responsive, project-focused portfolio website for Greyson Alegria, a Computer Science student at Appalachian State University.

## Files

- `index.html` - Main page content and section structure
- `style.css` - Responsive styling and visual design
- `script.js` - Mobile navigation and dynamic footer year
- `assets/` - Place resume PDF, screenshots, and future portfolio assets here

## Update Before Publishing

In `index.html`, replace the placeholder `#` links for:

- Resume
- GitHub
- LinkedIn

Suggested resume path:

```text
assets/Greyson-Alegria-Resume.pdf
```

## Sections

- Hero
- Featured Projects
- Skills
- About
- Contact

## Run Locally

Open `index.html` directly in a browser. No build tools or dependencies are required.
