# Greyson Alegria Portfolio

A responsive portfolio for Greyson Alegria, a Computer Science student at Appalachian State University.

## Live Site

[greysona75.github.io/portfolio](https://greysona75.github.io/portfolio/)

## Featured Projects

- **ReAppraise** — SwiftUI prototype for photo-based resale and donation value estimates
- **Dulcet & Delish Cafe Website** — Responsive multi-page local business website
- **Bean** — SwiftUI coffee and matcha discovery app using OpenStreetMap and the Overpass API

## Site Content

- Project screenshots, technology stacks, and case studies
- Technical skills
- About and contact information
- Links to GitHub, LinkedIn, and the hosted resume

## Files

- `index.html` — Main portfolio content and metadata
- `style.css` — Responsive layout and visual design
- `script.js` — Mobile navigation and dynamic footer year
- `resume.html` — Fallback page linking to the hosted resume
- `assets/` — Project screenshots and favicon

## Run Locally

Open `index.html` in a browser. No build tools or dependencies are required.
